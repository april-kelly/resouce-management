Presentation

ABG_PhpToXls is a PHP class to generate Excel data from 2 dimensions PHP arrays in form of:

$PhpArray = array( array(CellA1, CellB1, ),

                   array(CellA2, CellB2, ),

                   . . . . . . . . .

                   array(CellAn, CellBn, )

                 );

 

Where CellXn data will be stored in the corresponding sheet cell.

Values accepted are:

Integer

: double word

Float

: 64 bits

Boolean

: converted to 0(false) and 1 (true)

Strings

: up to 65535 chars

Any other types are converted to Excel null cells

Generated Excel data may be sent to :

·   Disk file under specified path

·   Requesting user browser (via HTTP) for Opening/Downloading

In addition, a full set of checks is made on parameters to, via exception mechanism, inform user

Exceptions

Exceptions are raised during execution on faulty situations.

It is advisable to embed the class utilization into a try/catch structure like in the following code.

try{

  $PhpToXls = new ABG_PhpToXls($TestArray, null, 'Sample', true);

//    $PhpToXls->SendFile();

  $PhpToXls->SaveFile();

}

catch(Exception $Except){

  ABG_PhpToXls::ExceptPrint($Except);

}

 

This code will generate an Excel file (sample.xls) from the TestArray, in the directory of the script.

Alternatively, invoking SendFile() method will generate a Open/Download dialog in the user browser.
